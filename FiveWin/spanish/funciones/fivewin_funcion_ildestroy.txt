IlDestroy()
 Previous  Top  Next

Destroys an image list.

====== Sintaxis ======

 


 IlDestroy( <hImageList> ) --> lSuccess 


 


 
====== Parámetros ======


 <hImageList> The handle to the imagelist to be destroyed. 


 


 
====== Devuelve ======


 <lSuccess> Returns .t. if successful, or .f. otherwise.

Retrieves the current page margins.


====== Sintaxis ======





 PageGetMargins() --> aMargin 


 


 



====== Parámetros ======





 None. 


 



====== Devuelve ======





 <aMargin> An array containing { top, left, right, bottom } margins of
the printer page.

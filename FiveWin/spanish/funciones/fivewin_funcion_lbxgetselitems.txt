
====== LbxGetSelItems() ====== 
LbxGetSelItems()

Elements selected in a multiple selection listbox
====== Sintaxis ======
 LbxGetSelItems( <hWnd> ) --> anItems 
====== Parámetros ======
 <hWnd> Handle of the listbox 
====== Devuelve ======
 <anItems> Array that will contain the number of order of the elements selected in a listbox of multiple selection.

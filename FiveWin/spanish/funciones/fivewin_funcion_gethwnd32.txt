
====== GethWnd32() ======
GethWnd32()

To convert a window Handle from 16 to 32 bits
====== Sintaxis ======
 GethWnd32( <hWnd16> ) --> hWnd32 
====== Parámetros ======
 <hWnd16> Handle of window of 16 bits 
====== Devuelve ======
 <hWnd32> Handle of window of 32 bits 
====== Ejemplo ======
====== Código fuente ======
 SOURCE\WINAPI\CALDLL32.C

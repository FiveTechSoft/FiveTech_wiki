GetNMHDRIdFrom()
 Previous  Top  Next

Retrieves the identifier of the control sending a notify message.



====== Sintaxis ======





 GetNMHDRIdFrom( <nPtrNMHDR> ) --> nIdFrom 


 


 



====== Parámetros ======





 <nPtrNMHDR> A pointer to the notify message structure. 


 


 



====== Devuelve ======





 <nIdFrom> The control identifier field of the notify message structure.

 SOURCE\CLASSES\toolbar.prg

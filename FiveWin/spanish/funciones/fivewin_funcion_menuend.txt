
====== MenuEnd() ====== 
MenuEnd()

Ends a menu definition. Internally used.
====== Sintaxis ======
 MenuEnd() --> oMenu 
====== Parámetros ======
 None 
====== Devuelve ======
 <oMenu> The menu object already initialized and with all its submenus
built and initialized. 

Observations:

 This function is used automatically by FiveWin from the xBase commands.

 SAMPLES\TestMenu.prg

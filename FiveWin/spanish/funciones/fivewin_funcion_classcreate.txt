
====== ClassCreate() ======

ClassCreate()
Creation of a class for xBase++

====== Sintaxis ======

 ClassCreate( x, y, z, w ) --> object 

 Function to create a class. It is used for xBase++ since the original
name has more than 10 letters and xBase doesn't allow it. 

====== Parámetros ======

 <x> Pointer to the database Alias () 

 <y> Array with the address of the object from where it is executed the
call to the function ClassCreat 

 <z> Array with the names of the data of the object from where the call
is executed to the function ClassCreat 

 <w> Array with the names of the methods of the object from where the
call is executed to the function ClassCreat 

====== Devuelve ======

 <ClassCreate( x, y, z, w )> 

====== Ejemplo ======

 SOURCE\CLASSES\DATABASE.PRG 

====== Código fuente ======

 SOURCE\FUNCTION\DB10.PRG

SetGetColorFocus()


SetGetColorFocus()
 Previous  Top  Next

Activates Gets color change when focused and optionally sets the focus
color to use.



====== Sintaxis ======





 SetGetColorFocus( [<nClrFocus>] ) --> nil 


 


 



====== Parámetros ======



 


 


 <nClrFocus>   The optional color used when the Get is focused. 


 


 


 


 

====== Devuelve ======




   Nil.

 


 



====== Código fuente ======





 SOURCE\CLASSES\tget.prg 


 


 



====== Ejemplo ======



 


 


 SAMPLES\testget2.prg

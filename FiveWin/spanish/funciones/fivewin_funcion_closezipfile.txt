
====== CloseZipFile() ======

CloseZipFile()

 Previous  Top  Next

Closes the ZIP file

Syntax

 CloseZipFile() --> nil 

====== Parameters ======
 None 

====== Returns ======
 Nothing. 

 Closes the ZIP file previously in use calling GetFirstInZip() and
GetNextInZip() functions.

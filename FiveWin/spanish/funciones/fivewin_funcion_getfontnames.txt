GetFontNames()


GetFontNames()
 Previous  Top  Next

Returns an array with the fonts available on a specified device.

====== Sintaxis ======


 GetFontNames( <hDC> ) --> aFontName 


 

====== Parámetros ======


 <hDC> Handle to the device context. 


 

====== Devuelve ======


 <aFontName> An array with the names of the available fonts. 


 

====== Código fuente ======


 SOURCE\WINAPI\FONTS.C

   CreateFont() ChooseFont()

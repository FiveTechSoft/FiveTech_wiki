nMidiDevices()

Number of MIDI output present devices in the system
====== Sintaxis ======
 nMidiDevices() --> nDevs 
====== Parámetros ======
 None 
====== Devuelve ======
 <nDevs> It returns the number of MIDI output present devices in the one
system if the function is executed correctly.

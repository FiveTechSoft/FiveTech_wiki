SToD()
 Previous  Top  Next

Converts a date string in the format "YYYYMMDD" to a date value.

====== Sintaxis ======

 


 SToD( <cDate> ) --> dDate 


 


 
====== Parámetros ======


 <cDate> The date string to be converted. 


 


 
====== Devuelve ======


 <dDate> The result of the conversion.

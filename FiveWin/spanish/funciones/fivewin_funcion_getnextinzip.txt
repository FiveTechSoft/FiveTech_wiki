
====== GetNextInZip() ======

Retrieves info for the next file in the ZIP file

===============Syntax====================

 GetNextInZip()  \\ <aFileInfo>

==============Parameteres================

None

==============Returns =================

               must be called prior to use GetNextInZip():

    aFileInfo[ 5 ]	// File size
    aFileInfo[ 6 ]	// Compressed size

WSay()

 Previous  Top  Next




Write a text value into a window



====== Sintaxis ======





 WSay( <hWnd>, <hDC>, <nRow>, <nCol>, <cText>, <nClrFore>, <nClrBack>,
<hFont>, <lPixel>, <lTransparent>, <nAlign> ) --> nil 


 


 



====== Parámetros ======



 


 


 <hWnd> Handle of the window 


 


 


 


 <hDC> Device context 


 


 


 


 <nRow> Row 


 


 


 


 <nCol> Column 


 


 


 


 <cText> Text to draw 


 


 


 


 <nClrFore> Text color 


 


 


 


 <nClrBack> Back color 


 


 


 


 <hFont> Handle of the font 


 


 


 


 <lPixel> Meausures in pixels or in text mode 


 


 


 


 <lTransparent> Transparent background 


 


 


 


 <nAlign> Text align ( to see SetTextAlign() ) 


 


 



====== Devuelve ======


   Nil.

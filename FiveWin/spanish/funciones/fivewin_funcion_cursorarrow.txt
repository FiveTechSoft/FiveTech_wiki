
====== CursorArrow() ======

CursorArrow()

Select the Windows default arrow cursor
====== Sintaxis ======
 CursorArrow() --> nil 

 This function changes the mouse cursor to the traditional arrow in
window. 

====== Parámetros ======
 None. 
====== Devuelve ======
 Nothing.

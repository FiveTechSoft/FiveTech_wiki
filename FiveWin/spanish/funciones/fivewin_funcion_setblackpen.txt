====== SetBlackPen() ======

SetBlackPen() Selecciona el lapiz negro para pintar

====== Sintaxis ======

SetBlackPen( <hDC> ) --> hPrevPen 

====== Parámetros ======
^Nombre^Descripción^
|<hDC>| Identifica el manejador del dispositivo de contexto donde se seleccionara el lápiz negro. |

====== Devuelve ======
^Nombre^Descripción^
|<hPrevPen>| Identifica el manejador del lápiz seleccionado anteriormente. |


====== DelIniSection() ======

Borra una sección en un archivo INI

====== Sintaxis ======
 
 DelIniSection( <cSection>, <cFile> ) --> lSuccess  

====== Parámetros ======

| <cSection> | Sección  |
| <cFile> | Archivo ini  |

====== Retorno ======

|<lSuccess> | .t. si la operación tiene éxito |

====== Código Fuente ======

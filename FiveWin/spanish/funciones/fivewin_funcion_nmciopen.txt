
====== nMciOpen() ====== 
nMciOpen()

Initializes a device
====== Sintaxis ======
 nMciOpen( <cDeviceType>, <cElement> ) --> nId 
====== Parámetros ======
  <cDeviceType>  Name of the device type 
  <cElement>     Device-element name (usually a path). 
====== Devuelve ======
 <nId> Device identifier returned to the user.

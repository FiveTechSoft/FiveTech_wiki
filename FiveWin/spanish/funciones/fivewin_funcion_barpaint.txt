
====== BarPaint() ======

Dibuja la barra de botones

====== Sintaxis: ( Syntax: ) ======
 
 BarPaint( <hWnd>, <hDC> ) --> nil  

====== Parametros: ( Parameters: ) ======

 <hWnd> El manejador de la barra de botones.  

 <nDC> Es el manejador de contexto de la barra de botones

====== Retorno: ( Returns: ) ======

 Nil

SOURCE\CLASSES\BAR.PRG

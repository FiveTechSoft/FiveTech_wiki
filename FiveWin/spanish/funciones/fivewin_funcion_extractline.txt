Extract a line from a string starting at a given position.


====== Sintaxis ======





 ExtractLine( <cText>, @<nFrom> ) --> cLine 


 


 






====== Parámetros ======





 <cText> The string from which the line will be extracted. 


 


 


 


 <nFrom> The starting position inside cText from where the line will be
extracted. It has to be passed by reference as it is updated by the
function. 


 


 



====== Devuelve ======





 <cLine> The extracted line.

 Previous  Top  Next

Sends data to a socket.

====== Sintaxis ======


 SocketSend( <nSocket>, <cText> ) --> nBytesSent 


 

====== Parámetros ======


 <nSocket> Socket number 


 


 <cText> Data to be send 


 

====== Devuelve ======


 <nBytesSent> Total number of bytes sent

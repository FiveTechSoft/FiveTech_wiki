Retrieves the current number of rows of tabs in a tab control.

====== Sintaxis ======


 TabGetRowCount( <hWnd> ) --> nRow 


 

====== Parámetros ======


 <hWnd> Handle to the tab control. 


 


 
====== Devuelve ======


 <nRow> The number or rows.


====== PalBmpLoad() ====== 
PalBmpLoad()

Loads from resources a bitmap and their palette.
====== Sintaxis ======
 PalBmpLoad( <cResName> ) --> aBmpPal 
====== Parámetros ======
 <cResName> Name or identifier of the bitmap in the file of resources. 
====== Devuelve ======
 <aBmpPal> An array containing hBitmap and hPalette handles of the bitmap. 
====== Ejemplo ======
 SOURCE\CLASSES\BITMAP.PRG

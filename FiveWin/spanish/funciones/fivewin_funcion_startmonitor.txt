Start the resource monitor



====== Sintaxis ======





 StartMonitor() --> nil 


 


 





 The system uses resources to show the elements graphics. If these they
are not liberated appropiately they will take place lost. These lost ones
can cause the fall of the system. This function used next to EndMonitor()
she makes that the resources are liberated. 


 


 



====== Parámetros ======





 None 


 


 



====== Devuelve ======





 Nothing

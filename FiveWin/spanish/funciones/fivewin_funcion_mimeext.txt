Returns the mime type of a given extension.

====== Sintaxis ======


 MimeExt( <cExt> ) --> cType 


 

====== Parámetros ======


 <cExt> The file extension 


 


 
====== Devuelve ======


 <cType> The mime type. 


 

====== Ejemplo ======


 local cVar := MimeExt( "bmp" )


====== GetWndRect() ======
GetWndRect()
Retrieves the window all area coordinates
====== Sintaxis ======
 GetWndRect( <hWnd> ) --> aRect 
====== Parámetros ======
 <hWnd> The handle of the window whose rect area is to be retrieved. 
====== Devuelve ======
 <aRect> An array with the { nTop, nLeft, nBottom, nRight }

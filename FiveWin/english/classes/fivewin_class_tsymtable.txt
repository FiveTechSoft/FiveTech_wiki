====== Class TSymTable ======

Object oriented manipulation of Clipper internal symbol table                  
====== DATA ======
|None||

====== METHODS ======
^Method ^ Description                                    ^
|Get|Returns the symbol at a given position of the symbol table|
|GetFirst|Returns the first symbol of the table|
|GetLast|Returns the last symbol of the table|
|Len|Returns the number of symbols of the app symbol table|
|Size|Returns the physical size of the symbol table|

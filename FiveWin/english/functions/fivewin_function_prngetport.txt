====== PrnGetPort() ====== 



PrnGetPort()






Device name of the output port



====== Syntax ======





 PrnGetPort() --> cPort 


 


 



====== Parameters ======





 None 


 


 



====== Returns ======





 <cPort> Device name of the predetermined output port.

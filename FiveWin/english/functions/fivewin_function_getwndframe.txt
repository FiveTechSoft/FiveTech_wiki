====== GetWndFrame() ====== 



Returns the FiveWin main frame application window



====== Syntax ======





 GetWndFrame() --> oWndFrame 


 


 



====== Parameters ======





 None 


 


 



====== Returns ======





 <oWndFrame> Returns the Clipper object of the main MDI frame window.

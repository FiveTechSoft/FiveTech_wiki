====== nBmpColors() ======


It returns the number of colors of a Bitmap.

====== Syntax ======


 nBmpColors( <hPalette> ) --> nColors 


 

====== Parameters ======


 <hPalette> Handle of the bitmap. 


 

====== Returns ======


 <nColors> Number of colors of the bitmap.

====== FMimeDec() ======
De-encrypt a MIME file 

====== Syntax ====== 
fMimeDec(<cIn>,   <cOut> ) --> nil  

====== Parameters ====== 
^Parameter^Descripttion^
| <cIn> | Name of the file to be de-encrypted |
| <cOut> | Name of the file to which THE de-encrypted data to be written. |

====== Return ====== 
| Nil  |

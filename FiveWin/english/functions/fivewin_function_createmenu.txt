====== CreateMenu() ====== 
Creates an empty menu

====== Syntax ======
CreateMenu() --> hMenu 

====== Parameters ======
None 

====== Returns ======
^Returns^Description^
|<hMenu>| Handle of the new menu. Otherwise NULL |

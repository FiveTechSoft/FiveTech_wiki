====== nDbl2Flt() ======


Convert a Double to Float



====== Syntax ======





 nDbl2Flt( <nDouble> ) --> nFloat 


 


 



====== Parameters ======





 <nDouble> Number of double precision 


 


 



====== Returns ======





 <nFloat> Floating point num

====== Source Code ======





 SOURCE\FUNCTION\DBL2FLT.C

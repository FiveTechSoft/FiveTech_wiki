====== BarInvert() ======

To create the effect of pressing the button bar. 

====== Syntax: ======

BarInvert( <hWnd> ) --> NIL

====== Parameters: ======
^Parameter^Description^
|<hWnd>| Handle of the button bar.  |

====== Returns: ======

NIL

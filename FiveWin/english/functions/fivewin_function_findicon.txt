====== FindIcon() ======
Checks if an icon exist in the resources.

====== Syntax ====== 
FindIcon( <cIconName> ) --> lFound  

====== Parameters ====== 
^Parameter^Description^
| <cIconName>| The name of the icon to search |

====== Return ====== 
^Returns^Description^
| <lFound> |.T. if the icon was found in the resources.|

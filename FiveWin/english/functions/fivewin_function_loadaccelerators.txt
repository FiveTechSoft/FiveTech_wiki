====== LoadAccelerators() ====== 



LoadAccelerators()






Loads the specified accelerator table.



====== Syntax ======





 Loadaccelerators( <hInst>, <cAccName> ) --> hAccelerators 


 


 



====== Parameters ======





 <hInst> Indetifica an instance of the program that you this executing
that it contains the accelerators table to to load 


 


 





 <cAccName> Name from the table of accelerators to load. 


 


 



====== Returns ======





 <hAccelerators> Handle of the loaded accelerator table

====== Source Code ======





 SOURCE\WINAPI\ACCELERA.C


====== GetResources() ====== 
 



Retrieve the Handle of the file of resources



====== Syntax ======





 GetResources() --> hResources 


 


 



====== Parameters ======





 None 


 


 



====== Returns ======





 <hResources> Handle of the resources file

====== Source Code ======

====== RLCheck() ======


FiveWin Clipmore equivalent functions.

====== Syntax ======


 RLCheck( <hGlobalMem>, <nRecNo> ) --> lResult 


 

====== Parameters ======


 <hGlobalMem>    Pointer to global memory. 


 <nRecNo>       Record number to check. 


 

====== Returns ======


 <lResult>       The logical result of checking.

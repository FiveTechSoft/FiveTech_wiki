
======  GetSysFont() ====== 
 


Retrieves the Windows default font facename



====== Syntax ======





 GetSysFont() --> cWindowsFontFacename 


 


 



====== Parameters ======





 None 


 


 



====== Returns ======





 <cWindowsFonFaceName> Windows default font facename.

====== HelpIndex() ======

Shows the default help file index.  

====== Syntax: ======
 
 HelpIndex() --> nil
====== Parameters: ======

 none  
====== Returns: ======

 nothing  

====== Description: ======

HelpIndex() shows the main index of the default selected Help file.

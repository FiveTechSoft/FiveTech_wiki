
======  GetLogicalDrives() ====== 




Retrieves a bitmask representig the currently available disk drives.



====== Syntax ======





 GetLogicalDrives() --> nDrvMask 


 


 



====== Parameters ======




   None.

 


 




====== Returns ======





 <nDrvMask> The available drives bitmask or zero if the function fails.

====== TabCtrlAdd() ======
Add a page to tabstrip control 

====== Syntax ======

 TabCtrlAdd( <cPrompt> ) --> nil  

====== Parameters ======

| <cPrompt> | Label of the tabstrip page.|  

====== Returns ======

 nil

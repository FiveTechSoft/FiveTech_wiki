SetPrintDefault()
 Previous  Top  Next

Sets the specified printer as the default printer.

====== Syntax ======


 SetPrintDefault( <cModel> ) --> nil 


 

====== Parameters ======


 <cModel> The name of the new default printer. 


 

====== Returns ======


 Nil 


 

====== Source Code ======


 SOURCE\CLASSES\PRINTER.PRG

====== WndAdjRight() ====== 

WndAdjRight()

Align to the right of the window


====== Syntax ======


 WndAdjRight( <hWnd>, <hTop>, <hBottom> ) --> nil 


 


 



====== Parameters ======





 <hWnd> Handle of the control 


 


 


 


 <hTop> Handle of the oTop object 


 


 


 


 <hBottom> Handle of the oBottom object 


 


 



====== Returns ======





 Nothing

 SAMPLES\TESTALIGN.PRG

 SOURCE\WINAPI\WNDMOVE.C

====== JoyGetNumDevs() ======




Number of joystick devices supported by the system.



====== Syntax ======





 JoyGetNumDevs() --> nNumDevs 


 


 



====== Parameters ======





 None 


 


 



====== Returns ======





 <nNumDevs> Returns the number of joystick devices supported by the
joystick driver. If no driver is present, the function returns zero. 


 


 



====== Example ======





 SAMPLES\TESTJOY.PRG 


 


 



====== Source Code ======





 SOURCE\WINAPI\JOYSTICK.C

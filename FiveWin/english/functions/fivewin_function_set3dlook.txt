====== Set3DLook() ======



Set 3d look efect



====== Syntax ======





 Set3DLook( <lOnOff> ) --> nil 


 


 



====== Parameters ======





 <lOnOff> Flag to set or reset this efect. 


 


 



====== Returns ======





 None

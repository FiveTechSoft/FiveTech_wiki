====== SetActiveWindow() ======



Changes the current active window



====== Syntax ======





 SetActiveWindow( <hWnd> ) --> hPrevWnd 


 


 



====== Parameters ======





 <hWnd> Identifies the top level window to be activated 


 


 



====== Returns ======



 


 


 <hPrevWnd> The handle of the previous active window.

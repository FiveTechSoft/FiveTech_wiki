GetPromptRect()


GetPromptRect()
 Previous  Top  Next

Returns the coordinate of a prompt.

====== Syntax ======

 


 GetPromptRect( <hDC>, <cPrompt>, <hFont> ) --> aRect 


 


 
====== Parameters ======


 <hDC> The handle of the device context. 


 


 <cPrompt> The prompt. 


 


 <hFont> The handle of the font of the prompt. 


 


 
====== Returns ======


 <aRect> The coordinates of the prompt.

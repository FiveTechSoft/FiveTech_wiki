
**MsgRetryCancel()** 

Messages box with buttons Retry and Cancel.
  
**Syntax:** 

 MsgRetryCancel( <cMsg>, <cTitle> ) --> lRetry    
  
**Parameters:** 

| <cMsg> | Message to show into the box.|
| <cTitle> | Title of the messages box.|

  
**Returns:** 

<lRetry> This function returns .t. if the function ends pressing the button Retry and .f. if you presses Cancel.  
  
**Sample:** 

 SAMPLES\TESTMSG.PRG  
  
  
**Source code:** 

 SOURCE\FUNCTION\TEST.C

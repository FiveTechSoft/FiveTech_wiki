====== ShowCaret() ======


Shows the caret on the screen



====== Syntax ======





 ShowCaret( <hWnd> ) --> nil 


 


 



====== Parameters ======





 <hWnd> Handle of the window 


 


 



====== Returns ======





 Nothing

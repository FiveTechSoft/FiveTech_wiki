nHiByte()

 Previous  Top  Next




Returns the High Byte of a word (two bytes)



====== Syntax ======





 nHiByte( <nWord> ) --> nHiByte 


 


 



====== Parameters ======





 <nWord> A numeric (2 bytes) value 


 


 



====== Returns ======





 <nHiByte> A numeric (1 byte) value 


 


 



Observations:



 


 


 A word is made of: 


 


 


 


 WORD = LOW_BYTE + HIGH_BYTE

 SOURCE\\FUNCTION\\HiLoWord.c

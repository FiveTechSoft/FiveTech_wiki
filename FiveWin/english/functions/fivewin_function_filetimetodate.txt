FileTimeToDate()


FileTimeToDate()
 Previous  Top  Next

Converts a filetime value to a date value.

====== Syntax ======

 


 FileTimeToDate( <nFileTime> ) --> dDate 


 


 
====== Parameters ======


 <nFileTime> The filetime value to convert. 


 


 
====== Returns ======


 <dDate> The converted date value.

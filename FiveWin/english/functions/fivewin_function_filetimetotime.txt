FileTimeToTime()


FileTimeToTime()
 Previous  Top  Next

Converts a filetime value to a string time value.

====== Syntax ======

 


 FileTimeToDate( <nFileTime> ) --> cTime 


 


 
====== Parameters ======


 <nFileTime> The filetime value to convert. 


 


 
====== Returns ======


 <cTime> The converted string time value.

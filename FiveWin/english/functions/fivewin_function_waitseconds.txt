====== WaitSeconds() ====== 


WaitSeconds()


Waits the specified number of seconds.

====== Syntax ======


 WaitSeconds( <nSecs> ) --> nil 


 


====== Parameters ======


 <nSecs> The number of seconds to wait 


 

====== Example ======


 WaitSeconds( 2 )

====== DeleteMetafile() ======
Deleted the given metafile handle

====== Syntax ======
DeleteMetafile( <hMeta> ) --> lSuccess 

====== Parameters ======
^Parameter^Description^
|<hMeta>| Handle of metafile to be deleteted. |

====== Returns ======
^Returns^Description^
| <lSuccess> | .T. if the operation is successful |

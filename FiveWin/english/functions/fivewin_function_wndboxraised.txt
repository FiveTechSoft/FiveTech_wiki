====== WndBoxRaised() ====== 

WndBoxRaised()

Draws a box raised


====== Syntax ======

 WndBoxRaised( <hDC>, <nTop>, <nLeft>, <nBottom>, <nRight> ) --> nil 


 


 



====== Parameters ======





 <hDC> Identifies the handle of the device context. 


 


 


 


 <nTop>, <nLeft>, <nBottom>,<nRight> The coordinates of the box to draw. 


 


 



====== Returns ======





 Nil 


 


 



====== Example ======





 SAMPLES\\ToolTip.prg

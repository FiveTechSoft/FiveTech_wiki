====== WndWidth() ====== 



WndWidth()

Update the window width


====== Syntax ======


 WndWidth( <hWnd> [,<nNewWidth> ] ) --> nWidth 


 


 



====== Parameters ======



 


 


 <hWnd> Window handle 


 


 


 


 <nNewWidth> New width 


 


 



====== Returns ======





 <nWidth> If <nNewWidth> is nil return the current Width

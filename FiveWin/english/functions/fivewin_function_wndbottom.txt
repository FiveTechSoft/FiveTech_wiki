====== WndBottom() ====== 

WndBottom()

Resize the window height and update the bottom coordinate


====== Syntax ======


 WndBottom( <hWnd> [,<nNewBottom> ] ) --> nNewBottom 


 


 



====== Parameters ======



 


 


 <hWnd> Handle of the window 


 


 


 


 <nNewBottom> New height 


 


 



====== Returns ======





 <nNewBottom> New height

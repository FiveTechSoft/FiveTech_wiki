====== lMkDir() ====== 



lMkDir()





Creates a new directory



====== Syntax ======





 lMkDir( <cNewDir> ) --> lSuccess 


 


 



====== Parameters ======





 <cNewDir> The path of the new dir to create. 


 


 



====== Returns ======





 <lSuccess> If the directory was created successfully. 


 


 



====== Example ======





 lMkDir( "c:\\Customer" )

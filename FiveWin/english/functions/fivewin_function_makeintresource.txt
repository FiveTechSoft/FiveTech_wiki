MakeIntResource()


MakeIntResource()
 Previous  Top  Next

Converts an integer value to a resource type compatible with the
resource-management functions. This function is used in place of a string
containing the name of the resource.

====== Syntax ======

 


 MakeIntResource( <nValue> ) --> nResult 


 


 
====== Parameters ======


 <nValue> Specifies the integer value to be converted. 


 


 
====== Returns ======


 <nResult> The resource type value. 


 


 
Sample :

 


 


 


 
Source code:

 


 SOURCE\FUNCTION\MAKEINTR.C

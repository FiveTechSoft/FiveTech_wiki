Create new system caret


====== Syntax ======      

CreateCaret(<hWnd>   ,; <hBitmap>,; <nWidth> ,; <nHeight> )

====== Parameters ====== 

| <hWnd> | Handle of window. |
| <hBitmap> | Handle of bitmap to the new caret.  nil to solid caret and 1 to gray caret. |
| <nWidth> | Width of bitmap. |
| <nHeight> | Height of bitmap. |

====== Returns ======        

nil

StrChar()

 Previous  Top  Next




Obtain a character of a string



====== Syntax ======





 StrChar( <cString>, <nChar> ) --> cChar 


 


 



====== Parameters ======





 <cString> String of characters 


 


 


 


 <nChar> Position of a character within a character string 


 


 



====== Returns ======





 <cChar> Character returned

====== SetCursor() ======


Changes the Windows active cursor



====== Syntax ======





 SetCursor( <hCursor> ) --> hOldCursor 


 


 



====== Parameters ======





 <hCursor> The handle of the new mouse cursor. 


 


 



====== Returns ======





 <hOldCursor> The handle of the previous in use cursor. 


 


 



====== Example ======





 SAMPLES\\TestCurs.prg

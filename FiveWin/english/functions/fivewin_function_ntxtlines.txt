====== nTxtLines() ======



It calculates the number of lines of a file



====== Syntax ======





 nTxtLines( <cFileName> ) --> nLines 


 


 



====== Parameters ======





 <cFileName> File name 


 


 



====== Returns ======





 <nLines> Number of lines of a file.

Str2Hex()


Str2Hex()
 Previous  Top  Next

Converts the characters of a string in their hexadecimal representation.

====== Syntax ======

 


 Str2Hex( <cString> ) --> cHex 


 


 
====== Parameters ======


 <cString> The string to convert. 


 


 
====== Returns ======


 <cHex> The hexadecimal representation of the string. 


 


 
Sample :

 


 

 SAMPLES\TESTHEX2.PRG 


 

 


 
Source code:

 


 SOURCE\FUNCTION\STR2HEX.C


======  GetWinVer() ====== 




Returns the Windows version.

====== Syntax ======

 


 GetWinVer() --> aVersion 


 


 
====== Parameters ======


 None. 


 


 
====== Returns ======


 <aVersion> An array containing major and minor Windows version numbers. 


 


 
Sample :

 


 


 


 
Source code:

 


 SOURCE\WINAPI\WNDIS.C

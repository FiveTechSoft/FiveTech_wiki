Path and executable file from where you position a module



====== Syntax ======





 GetModuleFileName( <hInstance> ) --> cAppName 


 


 



====== Parameters ======





 <hInstance> Handle of the module 


 


 



====== Returns ======





 <cAppName> Path and executable file from where you position a module

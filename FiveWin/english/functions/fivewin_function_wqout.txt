WQOut()

FiveWin QOut() replacement


====== Syntax ======


 WQOut( <aParams> ) --> nil 


====== Parameters ======





 <aParams> An array containing all the values to be displayed on the
screen. 


 


 



====== Returns ======





 nil

 SAMPLES\\TestBVbx.prg

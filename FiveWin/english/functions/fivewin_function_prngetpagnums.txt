====== PrnGetPagNums() ====== 


PrnGetPagNums()


Retrieves the status of the Pages radiobutton of the print dialog.

====== Syntax ======


 PrnGetPagNums() -?> lPageNums 


 

====== Parameters ======


 None 


 

====== Returns ======


 <lPageNums> True if the Pages radiobutton of the print dialog is
selected, or false otherwise.

====== Source Code ======

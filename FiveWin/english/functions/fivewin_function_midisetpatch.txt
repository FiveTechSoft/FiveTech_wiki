====== MidiSetPatch() ======


Activates a patch.

====== Syntax ======

 


 MidiSetPatch( <hMidi>, <nChannel>, <nVoice> ) --> nReturn 


 


 
====== Parameters ======


 <hMidi>    Handle of the audio device driver 


 


 <nChannel> Channel number. 


 


 

 <nVoice>    Voice number. 


 


 
====== Returns ======


 <nReturn> Returned value by the audio device driver.

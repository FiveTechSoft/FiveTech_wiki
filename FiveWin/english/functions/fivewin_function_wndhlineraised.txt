====== WndHlineRaised() ====== 

WndHlineRaised()

Draw a horizontal line with raised look

====== Syntax ======

 WndHLineRaised( <hDC>, <nTop>, <nLeft>, <nBottom>, <nRight> ) --> nil 


 


 



====== Parameters ======



 


 


 <hDC> Device context 


 


 


 


 <nTop> Superior coordinate 


 


 


 


 <nLeft> Left coordinate 


 


 


 


 <nBottom> Inferior coordinate 


 


 


 


 <nRight> Right coordinate 


 


 



====== Returns ======





 Nothing

====== Source Code ======





 SOURCE\FUNCTION\WNDLINES.C

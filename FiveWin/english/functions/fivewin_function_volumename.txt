====== VolumeName() ====== 
Returns the name of a volume. 

====== Syntax ====== 

 VirtualRdd( [<cDriveRoot>] ) --> cName  
 
====== Parameters ====== 

|<cDrive>| The drive root (defaults to "C:\" ).|  
 
====== Returns ====== 

|<cName>| The name of the volume.|

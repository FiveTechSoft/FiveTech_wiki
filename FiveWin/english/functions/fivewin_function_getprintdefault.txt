GetPrintDefault()


GetPrintDefault()
 Previous  Top  Next

Returns the handle of device context of the default printer.

====== Syntax ======

 


 GetPrintDefault() --> hDC 


 


 
====== Parameters ======


 None. 


 


 
====== Returns ======


 <hDC>      The handle of the just created device context of the default
printer. 


 


 
Sample :

 


 


 


 
Source code:

 


 SOURCE\WINAPI\PRINTDC.C

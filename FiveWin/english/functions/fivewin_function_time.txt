
====== Time() ======

Gives the current time in format 00:00:00

====== Syntax ======





 Time() --> cTime 


 


 



====== Parameters ======





 None 


 


 



====== Returns ======



 


 


 <cTime> String of characters in format "00:00:00" hours, minutes,
seconds.

====== Source Code ======

====== FMimeEnc() ======
Encrypt a MIME file 

====== Syntax ====== 
fMimeEnc(<cIn>, <cOut> ) --> nil 

====== Parameters ====== 
^Parameter^Description^
| <cIn> | Name from the file to encrypt.  |
| <cOut> | Name of the encrypted file   |

====== Return ====== 
Nil

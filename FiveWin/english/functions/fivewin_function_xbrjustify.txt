====== xBrJustify() ======

Sets the alignments of the column of a xBrowse object. 

====== Syntax ======

 xBrJustify( <oBrw>, <aJustify> ) --> oBrw  
 
====== Parameters ======

|<oBrw>| The xBrowse object.|  
|<aJustify>| An array containing the alignment setting for each column.|  
 
====== Returns ====== 

|<oBrw>| The xBrowse object itself.|

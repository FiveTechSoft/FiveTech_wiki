Deletes an icon from the taskbar. 

====== Syntax ====== 

 TaskDelIcon( <oNotifyIconData> ) --> nil  
 
====== Parameters ====== 

|<oNotifyIconData>| The handle of the taskbar icon to delete.|  
 
====== Returns ====== 

 Nil

ReleaseCapture()

 Previous  Top  Next




Let the other applications use the mouse



====== Syntax ======





 ReleaseCapture() --> nil 


 


 



====== Parameters ======





 None 


 


 



====== Returns ======



 


 


 nil 


 


 



Observations:





 This function releases the mouse capture and restores normal input
processing.

 SAMPLES\\NewHelp.prg

====== SetAccTable() ======


Sets the accelerator table.

====== Syntax ======

 


 SetAccTable( <aItems> ) --> hAccel 


 


 
====== Parameters ======


 <aItems> The array of the accelerators to be defined. 


 


 
====== Returns ======


 <hAccel> The handle to the accelerator table.

Source code:

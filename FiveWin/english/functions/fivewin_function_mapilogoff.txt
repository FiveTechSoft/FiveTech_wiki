MAPILogoff()
 Previous  Top  Next

Ends a session with the messaging system.

====== Syntax ======

 


 MAPILogoff() --> lSuccess 


 


 
====== Parameters ======


 None. 


 


 
====== Returns ======


 <lSuccess> .t. if the function is successful, otherwise .f..

MAPIResolveName()


MAPIResolveName()
 Previous  Top  Next

Transforms a message recipient's name as entered by a user to an
unambiguous address list entry.

====== Syntax ======

 


 MAPIResolveName( [<cName>] ) --> cAddress 


 


 
====== Parameters ======


 <cName> The name to be resolved. 


 


 
====== Returns ======


 <cAddress> The address of the resolved name.

Source code:

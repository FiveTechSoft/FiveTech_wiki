
====== GetTimePick() ====== 


Retrieves the currently selected time from a date and time picker control.



====== Syntax ======





 GetTimePick( <hWnd> ) --> cTime 


 


 



====== Parameters ======





 <hWnd> The handle of the date and time picker control. 


 


 



====== Returns ======





 <cTime> The currently selected time.

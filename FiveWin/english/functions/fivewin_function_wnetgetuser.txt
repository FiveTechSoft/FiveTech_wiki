====== WNetGetUser() ====== 


WNetGetUser()

Retrieves the actual local Net user


====== Syntax ======


 WNetGetUser() --> cUserName 


====== Parameters ======





 None 


 


 



====== Returns ======





 <cUserName> The name of the current station user.

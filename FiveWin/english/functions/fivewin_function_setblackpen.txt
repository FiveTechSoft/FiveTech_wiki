====== SetBlackPen() ======



Selects the black pen for painting



====== Syntax ======





 SetBlackPen( <hDC> ) --> hPrevPen 


 


 



====== Parameters ======





 <hDC> Identifies the handle of the device context where select a black
pen. 


 


 



====== Returns ======





 <hPrevPen> Identifies the handle of the previous selected pen.

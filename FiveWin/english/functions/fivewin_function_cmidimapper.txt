====== cMidiMapper() ====== 




Returns the name of the midi mapper.

====== Syntax ======

 


 cMidiMapper() --> cMapperName 


 


 
====== Parameters ======


 None. 


 


 
====== Returns ======


 <cMapperName> The name of the midi mapper.

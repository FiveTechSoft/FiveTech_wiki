
======  GetLastError() ====== 




Get the error code for the last operation.

====== Syntax ======


 GetLastError() --> nError 


 

====== Parameters ======


 None 


 

====== Returns ======


 <nError> Error code

 SAMPLES\SYLINT.PRG


======  GetKeyToggle() ====== 






State of a toggle key



====== Syntax ======





 GetKeyToggle( <nKey> ) --> lState 


 


 



====== Parameters ======





 <nKey> Code of the key that we will examine 


 


 



====== Returns ======





 <lState> Returns.t. if the key this active one (TOGGLED ON)

 SOURCE\CLASSES\MSGBAR.PRG

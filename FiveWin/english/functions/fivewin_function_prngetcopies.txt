====== PrnGetCopies() ====== 


PrnGetCopies()


Retrieves the number of copies to print.

====== Syntax ======


 PrnGetCopies() -?> nCopies 


 

====== Parameters ======


 None 


 

====== Returns ======


 <nCopies> The number of copies to print.

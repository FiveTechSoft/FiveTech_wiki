====== CreatePopUpMenu() ====== 
Creates an empty pop-up menu

====== Syntax ======
CreatePopUpMenu() --> hMenu 

====== Parameters ======
None 

====== Returns ======
^Returns^Description^
|<hMenu>| Handle of the new popup menu|


====== uValBlank() ======

Returns the correspondent blank value of any variable 

====== Syntax: ======

 uValBlank( <uValue> ) --> uBlankValue  

====== Parameters: ======

|<uValue>| A variable whose value is to be blanked. | 
 
====== Returns: ======

|<uBlankValue>| The same value but blanked.  |
